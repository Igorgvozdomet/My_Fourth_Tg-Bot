# Telegram Бот — Конвертер валют

Telegram-бот на Python для конвертации валют. Поддерживает готовые пары и пользовательский ввод.

## Функционал

| Команда / Действие | Что делает |
|-------------------|-------------|
| `/start` | Запуск бота, ввод суммы |
| Выбор суммы | Показывает кнопки с популярными парами |
| USD/EUR, EUR/USD, USD/GBP, GBP/USD | Мгновенная конвертация |
| Другие значения | Ввод своей пары (например, `RUB/TRY`) |

## Технологии

- Python
- pyTelegramBotAPI
- CurrencyConverter
